This folder contains solutions for Task 1 and Task 2 from the FoCP assignment. Both programs are written in Python.

---

Task 1 – Pizza Pricing

This program calculates the total cost of a pizza order for Beckett Pizza Plaza, which offers a four-for-three deal.

Features
- Prompts the user to enter four pizza prices
- Rejects invalid input (non-numeric, zero, or negative values)
- Applies the discount by making the cheapest pizza free
- Displays:
  - The final order total
  - The percentage discount achieved

Key Concepts Used
- Lists
- Input validation
- Minimum value calculation
- Formatted output

---

Task 2 – Password Security Check

This program performs a simple password verification process.

Features
- Prompts the user to enter a password
- Checks that the password is at least 9 characters long
- If valid, asks the user to confirm three randomly chosen characters
- Exits immediately if any check fails
- Confirms success if all checks pass

Key Concepts Used
- String indexing
- Random number generation
- Conditional statements
- Loop control and early exit
